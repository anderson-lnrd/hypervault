Datepicker

Seletor de data com popup e calendario standalone. Suporta min/max, filtro, cores e navegacao por teclado.

Basico

Datepicker padrao com popup de calendario.

Data selecionada: {{ formatDate(selectedDate()) }} 

<hyper-datepicker (dateChange)="onDate($event)" />

Placeholder Customizado

Use placeholder para alterar o texto.

<hyper-datepicker placeholder="Data de nascimento" />

Cor Secondary

Use color="secondary" para trocar a cor.

<hyper-datepicker color="secondary" />

Min / Max

Restrinja o intervalo de datas com min e max.

<hyper-datepicker [min]="minDate" [max]="maxDate" />

Filtro de Data

Use dateFilter para desabilitar datas especificas (ex: sem fins de semana).

noWeekends = (date: Date): boolean => {
  const day = date.getDay();
  return day !== 0 && day !== 6;
};

<hyper-datepicker [dateFilter]="noWeekends" />

Desabilitado

Use [disabled]="true" para desabilitar o datepicker.

<hyper-datepicker [disabled]="true" />

Calendario Standalone

Use hyper-calendar diretamente como componente.

Selecionada: {{ formatDate(calendarDate()) }} 

<hyper-calendar (selectedChange)="onDate($event)" />
