Switch

Toggle switch com suporte a ControlValueAccessor, cores e estados. Ideal para configuracoes on/off.

Basico

Switch simples com label via content projection e two-way binding.

Notificacoes: {{ notifications ? 'ON' : 'OFF' }} Modo Offline 
<hyper-switch [(checked)]="notifications">
  Notificacoes: {{'{{' }} notifications ? 'ON' : 'OFF' {{ '}}' }}
</hyper-switch>

Cores

Cores disponiveis: primary (padrao) e secondary.

Primary Secondary 
<hyper-switch color="secondary">Modo Turbo</hyper-switch>

Desabilitado

Use [disabled]="true" para impedir interacao.

Desabilitado Checked Disabled 
<hyper-switch [disabled]="true">Desabilitado</hyper-switch>
