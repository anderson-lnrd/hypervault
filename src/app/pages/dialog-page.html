Dialog

Servico imperativo para abrir modais com backdrop, focus trap, ESC e injecao de dados.

Basico

Abre um dialog de confirmacao. O resultado retorna via afterClosed().

@if (result) {

{{ result }}

} 
const ref = this.dialog.open(MyDialogComponent);
ref.afterClosed().subscribe(result => console.log(result));

Com Dados

Passe dados via config.data e injete com HYPER_DIALOG_DATA.

this.dialog.open(MyComponent, {
  data: { title: 'Titulo', message: 'Mensagem' },
  width: '400px',
});

// No componente:
readonly data = inject(HYPER_DIALOG_DATA);

Tamanho Customizado

Configure width, maxWidth, height e maxHeight.

this.dialog.open(Comp, { width: '700px', maxHeight: '300px' });

Disable Close

Com disableClose: true, o dialog so fecha via botao interno.

this.dialog.open(Comp, { disableClose: true });

Estrutura do Dialog

Use as diretivas de layout dentro do componente do dialog.

@Component({
  imports: [HyperDialogTitle, HyperDialogContent, HyperDialogActions, HyperButton],
  template: \`
    <hyper-dialog-title>Titulo</hyper-dialog-title>
    <hyper-dialog-content>
      <p>Conteudo do dialog.</p>
    </hyper-dialog-content>
    <hyper-dialog-actions>
      <button hyper-stroked-button (click)="dialogRef.close()">Cancelar</button>
      <button hyper-raised-button (click)="dialogRef.close('ok')">OK</button>
    </hyper-dialog-actions>
  \`,
})
export class MyDialog {
  readonly dialogRef = inject(HyperDialogRef);
}
