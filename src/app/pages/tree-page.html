Tree

Componente de arvore hierarquica com expansao, icones, niveis e variantes visuais.

Basico

Arvore simples com nos expandiveis.

Frutas Maca Banana Laranja Vegetais Cenoura Brocolis 
<hyper-tree>
  <hyper-tree-node [expandable]="true" [level]="0">
    Frutas
    <hyper-tree-node [level]="1">Maca</hyper-tree-node>
  </hyper-tree-node>
</hyper-tree>

Com Icones

Adicione icones com o input icon.

Documentos Relatorio.pdf Planilha.xlsx Fotos foto1.jpg foto2.png Musicas Track 01.mp3 Track 02.mp3 
<hyper-tree-node icon="folder" [expandable]="true">...</hyper-tree-node>

Niveis Profundos

Suporte a aninhamento profundo com indentacao automatica.

Brasil Sao Paulo Capital Paulista Pinheiros Vila Mariana Campinas Rio de Janeiro Niteroi 
<hyper-tree-node [level]="3" icon="place">Paulista</hyper-tree-node>

Dense

Use [dense]="true" para um layout mais compacto.

src app app.ts app.html app.scss main.ts index.html 
<hyper-tree [dense]="true">...</hyper-tree>

Bordered

Use [bordered]="true" para adicionar bordas entre nos.

Configuracoes Perfil Seguranca Notificacoes Sobre Licenca Versao 
<hyper-tree [bordered]="true">...</hyper-tree>

Expand / Collapse All

Use expandAll() e collapseAll() para controlar todos os nos.

Categorias Roupas Camisetas Calcas Eletronicos Celulares Notebooks 
@ViewChild('tree') tree!: HyperTree;
tree.expandAll();
tree.collapseAll();

Desabilitado

Use [disabled]="true" para desabilitar nos individuais.

Projeto Arquivo bloqueado Arquivo acessivel 
<hyper-tree-node [disabled]="true">Bloqueado</hyper-tree-node>
