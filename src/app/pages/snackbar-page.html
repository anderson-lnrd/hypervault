Snackbar

Notificacoes temporarias via servico imperativo. Suporta variantes, posicoes, icones, acoes e auto-dismiss.

Basico

Abra um snackbar simples com HyperSnackbar.open().

snackbar.open({{ '{' }} message: 'Operacao realizada.' {{ '}' }});

Variantes

Atalhos semanticos: success(), error() e warning(). Cada um inclui icone padrao.

snackbar.success('Salvo com sucesso!');
snackbar.error('Falha ao conectar ao servidor.');
snackbar.warning('Sua sessao expira em 5 minutos.');

Com Acao

Adicione um botao de acao com action. Use afterDismissed() para saber se a acao foi clicada.

const ref = snackbar.open({{ '{' }}
  message: 'Item removido.',
  action: 'Desfazer',
  duration: 6000,
{{ '}' }});

ref.afterDismissed().then(actionClicked => {{ '{' }}
  if (actionClicked) snackbar.success('Item restaurado!');
{{ '}' }});

Com Icone

Exiba um Material Icon com a propriedade icon.

snackbar.open({{ '{' }}
  message: 'Novo email recebido!',
  icon: 'mail',
{{ '}' }});

Posicoes

Seis posicoes na tela: top-left, top-center, top-right, bottom-left, bottom-center (padrao), bottom-right.

snackbar.open({{ '{' }}
  message: 'Notificacao no topo!',
  position: 'top-center',
{{ '}' }});

Duracao

Controle o tempo de auto-dismiss com duration (em ms). Use 0 para desabilitar o auto-dismiss.

// Auto-dismiss apos 10s
snackbar.open({{ '{' }} message: '...', duration: 10000 {{ '}' }});

// Sem auto-dismiss
snackbar.open({{ '{' }} message: '...', action: 'Fechar', duration: 0 {{ '}' }});
