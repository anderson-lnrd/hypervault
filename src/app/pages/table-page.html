Table

Tabela de dados com sorting, sticky header, variantes visuais e no-data row.

Basico

Tabela simples com header e rows.

ID {{ row.id }} Nome {{ row.name }} Cargo {{ row.role }} Nivel {{ row.level }} 
<hyper-table [dataSource]="data" [hyperHeaderRowDef]="cols" [hyperRowDef]="cols">
  <ng-container hyperColumnDef="name">
    <span *hyperHeaderCellDef>Nome</span>
    <span *hyperCellDef="let row">{{ row.name }}</span>
  </ng-container>
</hyper-table>

Bordered & Striped

Variantes visuais com [bordered] e [striped].

ID {{ row.id }} Nome {{ row.name }} Cargo {{ row.role }} Nivel {{ row.level }} 
<hyper-table [bordered]="true" [striped]="true" [dense]="true">...</hyper-table>

Sorting

Clique nos headers para ordenar. Ciclo: asc → desc → sem ordenacao.

ID {{ row.id }} Nome {{ row.name }} Cargo {{ row.role }} Nivel {{ row.level }} 
<span *hyperHeaderCellDef>
  <span hyper-sort-header="name">Nome</span>
</span>

(sortChange)="onSort($event)"

Com Paginator

Combine Table com Paginator para paginacao.

ID {{ row.id }} Nome {{ row.name }} Cargo {{ row.role }} Nivel {{ row.level }} 
<hyper-paginator [length]="total" [pageSize]="5" (page)="onPage($event)" />

No Data Row

Quando dataSource esta vazio, exibe conteudo fallback.

Nome {{ row.name }} Cargo {{ row.role }}
Nenhum dado encontrado.
 
<div hyperNoDataRow>Nenhum dado encontrado.</div>
