Menu

Context menu com icones, grupos, dividers, posicionamento e navegacao por teclado.

Basico

Clique no botao para abrir o menu. Itens emitem (activated) ao serem clicados.

@if (lastAction()) { > {{ lastAction() }} }
Copiar Colar Recortar 
<button [hyperMenuTriggerFor]="menu">Abrir</button>
<hyper-menu #menu>
  <hyper-menu-item (activated)="fn()">Copiar</hyper-menu-item>
</hyper-menu>

Com Icones

Use o input icon para adicionar icones Material aos itens.

Editar Duplicar Compartilhar Excluir Novo Abrir Salvar Exportar Imprimir 
<hyper-menu-item icon="edit">Editar</hyper-menu-item>
<hyper-menu-divider />
<hyper-menu-item icon="delete">Excluir</hyper-menu-item>

Grupos

Agrupe itens com hyper-menu-group e separadores para organizar o menu.

Perfil Preferencias Notificacoes Alterar Senha Sair 
<hyper-menu-group label="Conta">
  <hyper-menu-item icon="person">Perfil</hyper-menu-item>
</hyper-menu-group>
<hyper-menu-divider />

Itens Desabilitados

Use [disabled]="true" para desabilitar itens. Eles nao recebem hover, focus ou click.

Visualizar Editar Admin (sem permissao) Excluir (bloqueado) 
<hyper-menu-item [disabled]="true">Bloqueado</hyper-menu-item>

Posicionamento

Configure xPosition (before/after) e yPosition (above/below). O menu auto-ajusta se ultrapassar a viewport.

Opcao 1 Opcao 2 Opcao 3 Opcao A Opcao B Opcao C Opcao X Opcao Y Opcao Z Item I Item II Item III 
<hyper-menu yPosition="above" xPosition="before">
  ...
</hyper-menu>
