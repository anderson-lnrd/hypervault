Button

Componente de botao com multiplas variantes, tamanhos, estados e efeitos hover. Suporta icones, loading state e o sistema de densidade.

Variantes

Seis variantes de botao disponiveis, cada uma com identidade visual propria.

<button hyper-raised-button>Raised</button>
<button hyper-raised-button color="secondary">Secondary</button>
<button hyper-stroked-button>Stroked</button>
<button hyper-raised-button color="destructive">Destructive</button>
<button hyper-ghost-button>Ghost</button>
<button hyper-link-button>Link</button>

Cores

Cada variante aceita um input color com os valores: primary, secondary, accent e destructive.

<button hyper-raised-button color="primary">Primary</button>
<button hyper-raised-button color="secondary">Secondary</button>
<button hyper-raised-button color="accent">Accent</button>
<button hyper-raised-button color="destructive">Destructive</button>

Tamanhos

Tres tamanhos disponiveis: sm, md (padrao) e lg.

<button hyper-raised-button size="sm">Small</button>
<button hyper-raised-button>Default (md)</button>
<button hyper-raised-button size="lg">Large</button>

Com Icones

Use <hyper-icon> dentro do botao para adicionar icones Material Icons antes ou depois do texto.

<button hyper-raised-button>
  <hyper-icon>shopping_cart</hyper-icon>
  Adicionar
</button>

Icon Buttons

Botoes redondos apenas com icone. Use o seletor hyper-icon-button e sempre forneça aria-label.

<button hyper-icon-button color="primary" aria-label="Adicionar">
  <hyper-icon>add</hyper-icon>
</button>

Estados

Os inputs disabled e [loading] controlam o estado do botao. Loading exibe um spinner e desabilita automaticamente.

<button hyper-raised-button disabled>Desabilitado</button>
<button hyper-raised-button [loading]="true">Carregando...</button>

Hover Effects

Efeitos personalizados ao passar o mouse. Valores disponiveis: "float-3d", "press-3d", "hover-accent", "hover-primary" e "hover-destructive". Combine efeitos separados por espaco.

3D — Float vs Press

Troca de Cor

Combinados (3D + Cor)

<!-- Float 3D — flutua para cima -->
<button hyper-raised-button hoverEffect="float-3d">Float 3D</button>

<!-- Press 3D — pressiona para baixo -->
<button hyper-raised-button hoverEffect="press-3d">Press 3D</button>

<!-- Combinando 3D + troca de cor -->
<button hyper-raised-button hoverEffect="float-3d hover-accent">Float + Accent</button>
<button hyper-stroked-button hoverEffect="press-3d hover-destructive">Press + Destructive</button>

Density Scale

Sistema de densidade de 0 (mais espaçoso) a -4 (mais compacto). Aplique a diretiva [hyperDensity] no elemento pai.

@for (d of [0, -1, -2, -3, -4]; track d) {
density: {{ d }}
} 
<div [hyperDensity]="-2">
  <button hyper-raised-button>Compact</button>
</div>

API

Referencia completa dos seletores e inputs do componente.

Seletores

SeletorDescricao
button[hyper-button]Botao flat (sem sombra)
button[hyper-raised-button]Botao elevado com sombra brutal
button[hyper-stroked-button]Botao com borda e sombra, fundo transparente
button[hyper-icon-button]Botao quadrado para icones
button[hyper-ghost-button]Botao fantasma, sem borda nem sombra
button[hyper-link-button]Botao estilo hyperlink

Inputs

InputTipoDefaultDescricao
color'primary' | 'secondary' | 'accent' | 'destructive'—Cor do botao
size'sm' | 'md' | 'lg''md'Tamanho do botao
hoverEffect'none' | 'float-3d' | 'press-3d' | 'hover-accent' | 'hover-primary' | 'hover-destructive''none'Efeito ao passar o mouse. Combine valores separados por espaco.
loadingbooleanfalseExibe spinner e desabilita o botao
