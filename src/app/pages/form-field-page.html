Form Field

Wrapper para inputs com label, prefixo e sufixo. Fornece estilo consistente para campos de texto, email, senha e textarea.

Input Basico

Use a classe hyper-input no elemento <input> dentro de <hyper-form-field>.

<hyper-form-field label="Email">
  <input class="hyper-input" placeholder="seu@email.com" />
</hyper-form-field>

Com Prefixo (Icone)

Use a diretiva hyperPrefix em um <hyper-icon> para exibir um icone antes do input.

search mail 
<hyper-form-field>
  <hyper-icon hyperPrefix>search</hyper-icon>
  <input class="hyper-input" placeholder="Buscar..." />
</hyper-form-field>

Com Sufixo

A diretiva hyperSuffix posiciona um elemento apos o input. Ideal para botoes de toggle ou acoes.

<hyper-form-field label="Senha">
  <input class="hyper-input" [type]="show ? 'text' : 'password'" />
  <button hyperSuffix (click)="show = !show">
    <hyper-icon>visibility</hyper-icon>
  </button>
</hyper-form-field>

Textarea

Use a classe hyper-textarea no elemento <textarea>.

<hyper-form-field label="Mensagem">
  <textarea class="hyper-textarea" rows="4"></textarea>
</hyper-form-field>
