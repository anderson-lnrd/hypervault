List

Componente de lista com itens interativos, icones, meta info, dividers, subheaders e lista de selecao.

Basico

Lista simples com bordas.

Inbox Enviados Rascunhos Lixeira 
<hyper-list [bordered]="true">
  <hyper-list-item>Inbox</hyper-list-item>
  <hyper-list-item>Enviados</hyper-list-item>
</hyper-list>

Com Icones e Meta

Adicione icones (leading) e meta info (trailing) via content projection.

inbox Inbox 42 send Enviados 12 drafts Rascunhos 3 
<hyper-list-item>
  <span hyper-list-item-icon class="material-icons">inbox</span>
  Inbox
  <span hyper-list-item-meta>42</span>
</hyper-list-item>

Multi-line

Use hyper-list-item-line para itens com titulo e descricao.

person Maria Silva Desenvolvedora Frontend person Joao Santos Designer UI/UX 
<hyper-list-item>
  <span hyper-list-item-icon class="material-icons">person</span>
  <span hyper-list-item-line>Maria Silva</span>
  <span hyper-list-item-line>Desenvolvedora Frontend</span>
</hyper-list-item>

Dividers e Subheaders

Agrupe itens com subheaders e dividers.

Pastas folder Documentos folder Imagens Arquivos description relatorio.pdf image foto.jpg 
<hyper-list-subheader>Pastas</hyper-list-subheader>
<hyper-list-item>...</hyper-list-item>
<hyper-list-divider />
<hyper-list-subheader>Arquivos</hyper-list-subheader>

Dense

Use [dense]="true" para reduzir o padding dos itens.

Item compacto 1 Item compacto 2 Item compacto 3 Item compacto 4 
<hyper-list [bordered]="true" [dense]="true">...</hyper-list>

Desabilitado e Selecionado

Itens podem ser desabilitados ou marcados como selecionados.

check Selecionado Normal Desabilitado 
<hyper-list-item [selected]="true">Selecionado</hyper-list-item>
<hyper-list-item [disabled]="true">Desabilitado</hyper-list-item>

Selection List

Lista com checkboxes para selecao multipla.

Angular React Vue Svelte (disabled)

Selecionados: {{ selectedItems().join(', ') || 'nenhum' }}

<hyper-selection-list [bordered]="true" (selectionChange)="onChange($event)">
  <hyper-list-option value="angular">Angular</hyper-list-option>
  <hyper-list-option value="react">React</hyper-list-option>
</hyper-selection-list>
