Sidenav

Painel lateral de navegacao (drawer) com tres modos de exibicao, posicionamento start/end e controle programatico.

Dashboard Produtos Relatorios

Conteudo principal. Use os controles acima para abrir/fechar o sidenav e trocar modo e posicao.

Modos

Tres modos de exibicao controlam como o sidenav interage com o conteudo.

ModoComportamento
overAbre sobre o conteudo com backdrop escuro. Clique no backdrop para fechar.
pushEmpurra o conteudo para o lado. Backdrop visivel, clicavel para fechar.
sideFica ao lado do conteudo sem backdrop. Nao fecha ao clicar fora.

Exemplo de Uso

<hyper-sidenav-container>
  <hyper-sidenav [(opened)]="menuAberto" mode="side">
    <!-- conteudo do menu -->
  </hyper-sidenav>
  <hyper-sidenav-content>
    <!-- conteudo principal -->
  </hyper-sidenav-content>
</hyper-sidenav-container>

Controle Programatico

Acesse a instancia via viewChild para controlar programaticamente.

readonly sidenav = viewChild(HyperSidenav);

// Abrir
this.sidenav().open();

// Fechar
this.sidenav().close();

// Toggle
this.sidenav().toggle();
