Como Usar

Todas as cores estao disponiveis via CSS custom properties. Para criar um tema customizado, sobrescreva as variaveis em :root.

/* Sobrescrevendo cores no seu tema */
:root {{ '{' }}
  --primary: #00ffcc;
  --secondary: #ff6600;
  --background: #111;
  --foreground: #eee;
{{ '}' }}

/* Usando em CSS */
.my-element {{ '{' }}
  background: var(--primary);
  color: var(--primary-foreground);
  border: var(--border-width) solid var(--foreground);
  box-shadow: var(--shadow-offset) var(--shadow-offset) 0 var(--foreground);
{{ '}' }}
