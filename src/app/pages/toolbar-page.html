Toolbar

Barra de ferramentas flexivel com suporte a cores, tamanhos, densidade, multiplas linhas e integracao nativa com o Sidenav. Slots de layout organizam conteudo em start, center e end.

Basico

Toolbar simples com titulo e botoes de acao.

Meu App 
<hyper-toolbar>
  <button hyper-icon-button aria-label="Menu">
    <hyper-icon>menu</hyper-icon>
  </button>
  <span hyper-toolbar-title>Meu App</span>
  <hyper-toolbar-end>
    <button hyper-icon-button aria-label="Buscar">
      <hyper-icon>search</hyper-icon>
    </button>
  </hyper-toolbar-end>
</hyper-toolbar>

Cores

Quatro variantes de cor: default, primary, secondary e accent.

Default Primary Secondary Accent 
<hyper-toolbar color="primary">...</hyper-toolbar>
<hyper-toolbar color="secondary">...</hyper-toolbar>
<hyper-toolbar color="accent">...</hyper-toolbar>

Tamanhos

Tres tamanhos: sm, md (padrao) e lg.

Small Medium (default) Large 
<hyper-toolbar size="sm">...</hyper-toolbar>
<hyper-toolbar size="lg">...</hyper-toolbar>

Layout: Start / Center / End

Use as diretivas hyper-toolbar-start, hyper-toolbar-center e hyper-toolbar-end para organizar o conteudo.

Titulo Central 
<hyper-toolbar color="primary">
  <hyper-toolbar-start>
    <button hyper-icon-button><hyper-icon>menu</hyper-icon></button>
  </hyper-toolbar-start>
  <hyper-toolbar-center>
    <span hyper-toolbar-title>Titulo Central</span>
  </hyper-toolbar-center>
  <hyper-toolbar-end>
    <button hyper-icon-button><hyper-icon>notifications</hyper-icon></button>
    <button hyper-icon-button><hyper-icon>account_circle</hyper-icon></button>
  </hyper-toolbar-end>
</hyper-toolbar>

Separador

Use <hyper-toolbar-separator> para criar divisoes visuais entre grupos de acoes.

Dashboard 
<hyper-toolbar>
  <button hyper-icon-button>...</button>
  <hyper-toolbar-separator />
  <span hyper-toolbar-title>Dashboard</span>
  <hyper-toolbar-end>
    <button hyper-ghost-button>Exportar</button>
    <hyper-toolbar-separator />
    <button hyper-icon-button>...</button>
  </hyper-toolbar-end>
</hyper-toolbar>

Elevated (Sombra Brutal)

Adicione [elevated]="true" para aplicar sombra no estilo neo-brutalist.

Elevated Toolbar BETA 
<hyper-toolbar [elevated]="true" color="primary">
  <span hyper-toolbar-title>Elevated</span>
</hyper-toolbar>

Dense (Compacto)

[dense]="true" reduz padding e altura. Ideal para sub-toolbars ou interfaces compactas.

Normal Dense 
<hyper-toolbar [dense]="true">
  <span hyper-toolbar-title>Compact Toolbar</span>
</hyper-toolbar>

Multi-Row

Use <hyper-toolbar-row> para adicionar linhas extras. Util para sub-navegacao, filtros ou breadcrumbs.

Painel Admin 
<hyper-toolbar>
  <span hyper-toolbar-title>Admin</span>
  <hyper-toolbar-row>
    <button hyper-ghost-button size="sm">Usuarios</button>
    <button hyper-ghost-button size="sm">Relatorios</button>
  </hyper-toolbar-row>
</hyper-toolbar>

Sticky

[sticky]="true" fixa o toolbar no topo durante o scroll. Funciona dentro de qualquer container com overflow.

<hyper-toolbar [sticky]="true">
  <span hyper-toolbar-title>Fixed Header</span>
</hyper-toolbar>

Com Sidenav

O toolbar foi projetado para funcionar dentro de hyper-sidenav-content, ocupando a largura total do conteudo. O botao de menu alterna o sidenav.

<hyper-sidenav-container>
  <hyper-sidenav [(opened)]="opened" mode="side">
    ...nav...
  </hyper-sidenav>
  <hyper-sidenav-content>
    <hyper-toolbar [sticky]="true">
      <button hyper-icon-button (click)="opened = !opened">
        <hyper-icon>menu</hyper-icon>
      </button>
      <span hyper-toolbar-title>Dashboard</span>
    </hyper-toolbar>
    <main>...</main>
  </hyper-sidenav-content>
</hyper-sidenav-container>

Density Scale

O toolbar respeita o sistema de densidade. Aplique [hyperDensity] para ajustar de 0 (padrao) a -4 (mais compacto).

@for (d of [0, -1, -2, -3, -4]; track d) {
density: {{ d }} Toolbar
} 
<div [hyperDensity]="-2">
  <hyper-toolbar>...</hyper-toolbar>
</div>
