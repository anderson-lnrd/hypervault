Stepper

Componente de etapas para fluxos multi-step. Suporta horizontal, vertical, linear e estados de erro.

Basico

Stepper horizontal simples com 3 etapas.

Preencha seu nome e email.

Informe seu endereco completo.

Revise e confirme seus dados.

 
<hyper-stepper>
  <hyper-step label="Etapa 1">
    <button hyperStepperNext>Proximo</button>
  </hyper-step>
</hyper-stepper>

Com Icones

Adicione icones customizados com o input icon.

Revise os itens do carrinho.

Escolha a forma de pagamento.

Selecione o metodo de envio.

Pedido realizado com sucesso!

 
<hyper-step label="Carrinho" icon="shopping_cart">...</hyper-step>

Vertical

Use orientation="vertical" para layout em coluna.

Registre-se com email e senha.

Verifique o codigo enviado ao seu email.

Adicione foto e bio (opcional).

Sua conta esta configurada!

 
<hyper-stepper orientation="vertical">...</hyper-stepper>

Linear

Com [linear]="true", so avanca se o step anterior estiver completo.

Aceite os termos para continuar.

Preencha os dados obrigatorios.

Processo concluido.

 
<hyper-stepper [linear]="true">...</hyper-stepper>

Estado de Erro

Use [hasError]="true" e errorMessage para indicar problemas.

Simule um erro no formulario.

Revise as informacoes.

 
<hyper-step [hasError]="true" errorMessage="Campos obrigatorios">
