Progress

Indicadores de progresso linear e circular. Progress Bar para barras horizontais e Spinner para indicadores circulares SVG.

Progress Bar

Determinado

Modo determinate — exibe progresso com base no value (0–100).

{{ determinateValue() }}% 
<hyper-progress-bar [value]="45" />

Indeterminado

Modo indeterminate — animacao continua para operacoes sem progresso mensuravel.

<hyper-progress-bar mode="indeterminate" />

Buffer

Modo buffer — exibe progresso principal e buffer (ex: video buffering).

<hyper-progress-bar mode="buffer" [value]="45" [bufferValue]="70" />

Query

Modo query — animacao reversa para indicar que o sistema esta buscando dados.

<hyper-progress-bar mode="query" />

Cores

Cores disponiveis: primary (padrao), secondary, accent e destructive.

<hyper-progress-bar [value]="60" color="secondary" />
<hyper-progress-bar [value]="60" color="accent" />
<hyper-progress-bar [value]="60" color="destructive" />

Striped & Animated

Use [striped]="true" para listras e [animated]="true" para animar.

<hyper-progress-bar [value]="65" [striped]="true" [animated]="true" />

Spinner

Indeterminado

Modo indeterminate (padrao) — rotacao continua.

<hyper-spinner />
<hyper-spinner [diameter]="24" [strokeWidth]="3" />
<hyper-spinner [diameter]="60" [strokeWidth]="6" />

Determinado

Modo determinate — exibe progresso de 0 a 100 no arco.

<hyper-spinner mode="determinate" [value]="75" />

Cores

Cores: primary (padrao), secondary, accent, destructive e current (herda currentColor).

<hyper-spinner color="secondary" />
<hyper-spinner color="accent" />
<hyper-spinner color="destructive" />

<!-- herda cor do pai -->
<span style="color: purple;">
  <hyper-spinner color="current" />
</span>

Tamanhos Customizados

Use diameter e strokeWidth para controlar dimensoes.

<hyper-spinner [diameter]="16" [strokeWidth]="2" />
<hyper-spinner [diameter]="64" [strokeWidth]="6" />
<hyper-spinner [diameter]="96" [strokeWidth]="8" />
