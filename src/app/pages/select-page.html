Select

Componente dropdown select com suporte a ControlValueAccessor, selecao multipla, grupos e busca por teclado.

Basico

Select simples com placeholder.

Maca Banana Cereja Uva

Selecionado: {{ selectedFruit || 'nenhum' }}

<hyper-select placeholder="Selecione uma fruta" [(value)]="selected">
  <hyper-option value="apple">Maca</hyper-option>
  <hyper-option value="banana">Banana</hyper-option>
</hyper-select>

Selecao Multipla

Use [multiple]="true" para permitir multiplas selecoes.

Vermelho Verde Azul Amarelo Roxo

Selecionados: {{ selectedColors.join(', ') || 'nenhum' }}

<hyper-select placeholder="Cores favoritas" [multiple]="true">
  <hyper-option value="red">Vermelho</hyper-option>
  <hyper-option value="green">Verde</hyper-option>
</hyper-select>

Grupos (OptGroup)

Agrupe opcoes com hyper-optgroup.

Gato Leao Tigre Cachorro Lobo 
<hyper-select placeholder="Selecione um animal">
  <hyper-optgroup label="Felinos">
    <hyper-option value="cat">Gato</hyper-option>
    <hyper-option value="lion">Leao</hyper-option>
  </hyper-optgroup>
</hyper-select>

Desabilitado

Desabilite o select inteiro ou opcoes/grupos individuais.

Opcao A Habilitada Desabilitada 
<hyper-select [disabled]="true" placeholder="Disabled">...</hyper-select>
<hyper-option value="no" [disabled]="true">Desabilitada</hyper-option>

Required

Use [required]="true" para indicar campo obrigatorio (exibe asterisco).

Opcao 1 Opcao 2 
<hyper-select [required]="true" placeholder="Campo obrigatorio">
  <hyper-option value="opt1">Opcao 1</hyper-option>
</hyper-select>

Com Formularios

Implementa ControlValueAccessor, compativel com ngModel e Reactive Forms.

<hyper-select [(value)]="selected">...</hyper-select>

// Reactive Forms
<hyper-select [formControl]="myControl">
  <hyper-option value="a">A</hyper-option>
</hyper-select>
