Dark / Light

Toggle via servico ou com o botao no picker.

themeService = inject(HyperThemeService);
themeService.setDarkMode(true);
themeService.toggleDarkMode();

Alto Contraste

Auto-contraste garante acessibilidade WCAG recalculando foregrounds para maximo contraste, engrossando bordas e realcando focus rings.

Estado: {{ themeService.highContrast() ? 'Alto contraste ativo' : 'Contraste normal' }} 

// Auto-contraste calcula luminancia relativa (WCAG 2.1)
// e forca foregrounds para #000 ou #fff conforme necessario.
themeService.setHighContrast(true);
themeService.toggleHighContrast();
themeService.highContrast(); // signal<boolean>

// Quando ativo:
// - Foregrounds: recalculados para maximo contraste
// - --border-width: 4px → 6px
// - --shadow-offset: 4px → 6px
// - Focus ring: 3px solid var(--primary)
// - Classe .hyper-high-contrast no <html>

Uso Programatico

Use HyperThemeService para controlar temas via codigo.

@for (theme of themeService.themes(); track theme.name) { } 
themeService = inject(HyperThemeService);
themeService.setTheme('ocean');
themeService.themeName();     // signal<string>
themeService.darkMode();      // signal<boolean>
themeService.activeTheme();   // signal<HyperThemeDefinition>

Tema Customizado

Registre temas customizados em runtime com registerTheme().

import { HyperThemeService, HyperThemeDefinition } from 'hypervault';

const meuTema: HyperThemeDefinition = {
  name: 'meu-tema',
  label: 'Meu Tema',
  dark: {
    background: '#0d1117',
    foreground: '#c9d1d9',
    card: '#161b22',
    muted: '#21262d',
    'muted-foreground': '#8b949e',
    input: '#161b22',
    border: '#c9d1d9',
    primary: '#58a6ff',
    'primary-foreground': '#0d1117',
    secondary: '#f78166',
    'secondary-foreground': '#0d1117',
    accent: '#7ee787',
    'accent-foreground': '#0d1117',
    destructive: '#f85149',
    'destructive-foreground': '#c9d1d9',
  },
  light: {
    background: '#ffffff',
    foreground: '#24292f',
    card: '#ffffff',
    muted: '#f6f8fa',
    'muted-foreground': '#57606a',
    input: '#ffffff',
    border: '#24292f',
    primary: '#0969da',
    'primary-foreground': '#ffffff',
    secondary: '#cf222e',
    'secondary-foreground': '#ffffff',
    accent: '#1a7f37',
    'accent-foreground': '#ffffff',
    destructive: '#cf222e',
    'destructive-foreground': '#ffffff',
  },
};

themeService.registerTheme(meuTema);
themeService.setTheme('meu-tema');

CSS Variables

Todos os tokens sao CSS custom properties. Voce tambem pode definir temas puramente via CSS.

:root {
  --background: #0f0f0f;
  --foreground: #fafafa;
  --card: #1a1a1a;
  --muted: #2a2a2a;
  --muted-foreground: #a3a3a3;
  --input: #242424;
  --border: #fafafa;
  --primary: #39ff14;
  --primary-foreground: #0f0f0f;
  --secondary: #ff1493;
  --secondary-foreground: #0f0f0f;
  --accent: #ffff00;
  --accent-foreground: #0f0f0f;
  --destructive: #ff4444;
  --destructive-foreground: #fafafa;
}
