Tooltip

Diretiva de tooltip com posicionamento automatico, delay configuravel e flip de viewport.

Basico

Passe o mouse sobre os botoes para ver o tooltip.

info 
<button hyper-raised-button hyperTooltip="Salvar documento">Salvar</button>

Posicoes

Posicoes: above (padrao), below, left e right.

<button hyperTooltip="Abaixo" tooltipPosition="below">Below</button>

Delay

Configure tooltipShowDelay e tooltipHideDelay em milissegundos.

<button hyperTooltip="Delay" [tooltipShowDelay]="500">500ms</button>

Desabilitado

Use [tooltipDisabled]="true" para desabilitar o tooltip.

<button hyperTooltip="Texto" [tooltipDisabled]="true">Disabled</button>

API

InputTipoDefaultDescricao
hyperTooltipstring—Texto do tooltip (required)
tooltipPosition'above' | 'below' | 'left' | 'right''above'Posicao do tooltip
tooltipShowDelaynumber200Delay para exibir (ms)
tooltipHideDelaynumber0Delay para esconder (ms)
