Paginator

Controle de paginacao com seletor de itens por pagina, navegacao e range label.

Basico

Paginator com 50 itens, 10 por pagina.

@if (pageEvent(); as pe) {

pageIndex: {{ pe.pageIndex }}, pageSize: {{ pe.pageSize }}, prev: {{ pe.previousPageIndex }}

} 
<hyper-paginator [length]="50" [pageSize]="10" (page)="onPage($event)" />

First/Last Buttons

Com [showFirstLastButtons]="true" exibe botoes de ir para primeira/ultima pagina.

<hyper-paginator [showFirstLastButtons]="true" />

Sem Seletor de Tamanho

Use [hidePageSize]="true" para esconder o seletor.

<hyper-paginator [hidePageSize]="true" />

Desabilitado

Use [disabled]="true" para desabilitar toda interacao.

<hyper-paginator [disabled]="true" />
