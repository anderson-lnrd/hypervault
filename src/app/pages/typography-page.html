Tipografia

Sistema tipografico do HyperVault. Duas familias complementares: Space Grotesk para titulos e UI, Space Mono para codigo e dados.

Familias Tipograficas

Duas variaveis CSS controlam toda a tipografia.

--font-sans Space Grotesk — ABCDEFGHIJKLM abcdefghijklm 0123456789
--font-mono Space Mono — ABCDEFGHIJKLM abcdefghijklm 0123456789

Escala — Sans (Space Grotesk)

Usada em titulos, labels, botoes e elementos de interface.

2.5rem / 900 HYPERVAULT
1.5rem / 800 Section Title
1.1rem / 700 Card Title
0.875rem / 600 BUTTON LABEL
0.85rem / 400 Corpo de texto padrao para paragrafos e descricoes mais longas.
0.75rem / 400 Caption ou texto auxiliar

Escala — Mono (Space Mono)

Usada em codigo, badges, dados, labels de formulario e texto tecnico.

0.85rem / 400 Descricao tecnica ou label
0.8rem / 400 Corpo monospace para dados
0.75rem / 400 const vault = new HyperVault();
0.7rem / 400 Texto auxiliar menor

Como Usar

/* Aplicando tipografia */
.title {{ '{' }}
  font-family: var(--font-sans);
  font-weight: 900;
  text-transform: uppercase;
  letter-spacing: 0.04em;
{{ '}' }}

.code-block {{ '{' }}
  font-family: var(--font-mono);
  font-size: 0.75rem;
  color: var(--primary);
{{ '}' }}

/* Google Fonts (ja incluido no index.html) */
@import url('https://fonts.googleapis.com/css2?family=Space+Grotesk:wght@400;600;700;800;900&family=Space+Mono:wght@400;700&display=swap');
