Breadcrumb

Navegacao hierarquica que se integra automaticamente com o Angular Router. Plug and play: basta definir data: {{ '{' }} breadcrumb: 'Label' {{ '}' }} nas rotas.

Modo Automatico (Router)

Zero configuracao. O componente le a arvore de rotas ativas e monta o breadcrumb automaticamente. So precisa de data.breadcrumb nas rotas.

<!-- No template: apenas isso -->
<hyper-breadcrumb />

<!-- Nas rotas: -->
const routes: Routes = [
  {{ '{' }} path: 'produtos', data: {{ '{' }} breadcrumb: 'Produtos' {{ '}' }}, ... {{ '}' }},
  {{ '{' }} path: 'detalhes/:id', data: {{ '{' }} breadcrumb: 'Detalhes' {{ '}' }}, ... {{ '}' }},
];

Modo Manual

Passe [items] para controle total. Ideal para cenarios sem router ou com navegacao customizada.

<hyper-breadcrumb [items]="items" />

items: HyperBreadcrumbItem[] = [
  {{ '{' }} label: 'Home', url: '/', icon: 'home' {{ '}' }},
  {{ '{' }} label: 'Categorias', url: '/categorias' {{ '}' }},
  {{ '{' }} label: 'Perifericos', url: '/categorias/perifericos' {{ '}' }},
  {{ '{' }} label: 'Teclados', url: '/categorias/perifericos/teclados' {{ '}' }},
];

Separador Customizado

O separador padrao e ›. Use o input [separator] para trocar.

<hyper-breadcrumb [items]="items" separator=">" />
<hyper-breadcrumb [items]="items" separator="/" />
<hyper-breadcrumb [items]="items" separator="→" />

Labels Dinamicos (Resolvers)

Use uma funcao em data.breadcrumb para gerar labels dinamicos a partir de dados resolvidos.

const routes: Routes = [
  {{ '{' }}
    path: 'produto/:id',
    resolve: {{ '{' }} produto: produtoResolver {{ '}' }},
    data: {{ '{' }}
      breadcrumb: (data: Data) => data['produto'].nome
    {{ '}' }},
    component: ProdutoPage,
  {{ '}' }},
];

Home Global (Injection Token)

Configure um item "Home" global com HYPER_BREADCRUMB_HOME. Sera prepended automaticamente em todas as paginas.

import {{ '{' }} HYPER_BREADCRUMB_HOME {{ '}' }} from 'hypervault';

// No app.config.ts ou providers do modulo:
providers: [
  {{ '{' }}
    provide: HYPER_BREADCRUMB_HOME,
    useValue: {{ '{' }} label: 'Inicio', url: '/', icon: 'home' {{ '}' }}
  {{ '}' }}
]

Fallback: Route Title

Quando data.breadcrumb nao esta definido, o componente usa a propriedade title da rota como fallback. Isso valoriza quem ja configura rotas conforme a documentacao oficial do Angular.

const routes: Routes = [
  {{ '{' }}
    path: 'dashboard',
    title: 'Dashboard',  // <-- usado como breadcrumb automaticamente
    component: DashboardPage,
  {{ '}' }},
];

Navegacao por Ancora (Fragment)

Use a propriedade fragment nos items para criar breadcrumbs que navegam para secoes internas da pagina — pontos de ancoragem via id.

<hyper-breadcrumb [items]="anchorItems" />

anchorItems: HyperBreadcrumbItem[] = [
  {{ '{' }} label: 'Topo', url: '/breadcrumb', fragment: 'topo', icon: 'vertical_align_top' {{ '}' }},
  {{ '{' }} label: 'Modo Automatico', url: '/breadcrumb', fragment: 'modo-automatico' {{ '}' }},
  {{ '{' }} label: 'Separadores', url: '/breadcrumb', fragment: 'separadores' {{ '}' }},
  {{ '{' }} label: 'API', url: '/breadcrumb', fragment: 'api' {{ '}' }},
];

<!-- No template, use ids nas secoes: -->
<section id="modo-automatico">...</section>
<section id="separadores">...</section>
