Alert

Componente de alerta inline para feedback contextual. Suporta variantes de cor, estilos outlined/filled, icones e acoes.

Variantes

Quatro variantes semanticas: info (padrao), success, warning e error.

Info Este e um alerta informativo. Sucesso Operacao realizada com sucesso. Aviso Atencao: esta acao pode ter consequencias. Erro Algo deu errado. Tente novamente. 
<hyper-alert variant="info">
  <hyper-alert-title>Info</hyper-alert-title>
  Este e um alerta informativo.
</hyper-alert>

<hyper-alert variant="error">
  <hyper-alert-title>Erro</hyper-alert-title>
  Algo deu errado.
</hyper-alert>

Outlined

Estilo outlined — borda colorida com fundo transparente.

Info Outlined Alerta com borda e sem preenchimento. Success Outlined Confirmacao com estilo outlined. Warning Outlined Aviso com estilo outlined. Error Outlined Erro com estilo outlined. 
<hyper-alert variant="success" [outlined]="true">
  <hyper-alert-title>Success Outlined</hyper-alert-title>
  Confirmacao com estilo outlined.
</hyper-alert>

Filled

Estilo filled — fundo solido com texto contrastante.

Info Filled Alerta com fundo solido. Success Filled Confirmacao com fundo solido. Warning Filled Aviso com fundo solido. Error Filled Erro com fundo solido. 
<hyper-alert variant="error" [filled]="true">
  <hyper-alert-title>Error Filled</hyper-alert-title>
  Erro com fundo solido.
</hyper-alert>

Com Icone

Use <hyper-alert-icon> para projetar um icone customizado. Cada variante ja exibe um icone padrao.

lightbulb Dica Voce pode customizar o icone com qualquer Material Icon. rocket_launch Deploy Concluido Sua aplicacao foi publicada com sucesso. 
<hyper-alert variant="info">
  <hyper-alert-icon>
    <span class="material-icons">lightbulb</span>
  </hyper-alert-icon>
  <hyper-alert-title>Dica</hyper-alert-title>
  Voce pode customizar o icone.
</hyper-alert>

Com Acoes

Use <hyper-alert-actions> para adicionar botoes de acao ao alerta.

Atualizacao Disponivel Uma nova versao esta disponivel. Deseja atualizar agora? 
<hyper-alert variant="warning">
  <hyper-alert-title>Atualizacao Disponivel</hyper-alert-title>
  Uma nova versao esta disponivel.
  <hyper-alert-actions>
    <button>Atualizar</button>
    <button>Depois</button>
  </hyper-alert-actions>
</hyper-alert>

Dismissible

Use [dismissible]="true" para exibir um botao de fechar. O evento (dismissed) e emitido ao clicar.

@if (showDismissible()) { Dispensavel Clique no X para fechar este alerta. } @else {

Alerta dispensado.

} 
<hyper-alert variant="info" [dismissible]="true" (dismissed)="onDismiss()">
  <hyper-alert-title>Dispensavel</hyper-alert-title>
  Clique no X para fechar este alerta.
</hyper-alert>
