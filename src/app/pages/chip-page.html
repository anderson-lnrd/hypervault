Chip

Chips compactos para tags, filtros e selecao. Inclui chip-set com selecao e chip-input para entrada dinamica.

Basico

Chips simples com label.

<hyper-chip label="Angular" />
<hyper-chip label="React" />

Variantes

Variantes: filled (padrao), outlined e tonal.

<hyper-chip label="Filled" variant="filled" />
<hyper-chip label="Outlined" variant="outlined" />
<hyper-chip label="Tonal" variant="tonal" />

Cores

Cores: primary, secondary, accent e destructive.

<hyper-chip label="Secondary" color="secondary" />
<hyper-chip label="Accent" color="accent" variant="outlined" />

Com Icone

Adicione icones com o input icon.

<hyper-chip label="Favorito" icon="star" />
<hyper-chip label="Configuracao" icon="settings" variant="outlined" />

Removivel

Use [removable]="true" para exibir botao de remocao.

@for (item of removableItems(); track item) { } 
<hyper-chip label="Tag" [removable]="true" (removed)="onRemove()" />

Chip Set — Selecao

Use hyper-chip-set com selectionMode para selecao unica ou multipla.

Single

Multiple

<hyper-chip-set selectionMode="single" color="secondary">
  <hyper-chip label="A" [selectable]="true" />
  <hyper-chip label="B" [selectable]="true" />
</hyper-chip-set>

Chip Input

Entrada de chips dinamica com hyper-chip-input. Suporta separadores (Enter, virgula), paste e ControlValueAccessor.

<hyper-chip-input placeholder="Digite e pressione Enter..." [(value)]="tags" />

Chip Input — Opcoes

Limite maximo de chips, sem duplicatas, com add on blur.

<hyper-chip-input
  [maxChips]="5"
  [allowDuplicates]="false"
  [addOnBlur]="true"
  color="secondary"
/>

Desabilitado

Chips e chip-input podem ser desabilitados.

<hyper-chip label="Disabled" [disabled]="true" />
<hyper-chip-input [disabled]="true" />
